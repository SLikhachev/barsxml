# Январь 2026

## Добавили новые теги

### ZL_LIST/SCHET/PLAT
Кто платит по счету (СМО, ТФОМС), поскольку в один пакет я включаю все записи,
не разбивая по СМО, то тупа ставлю код 25011 - СМО ВС Альянс

### ZL_LIST/ZAP/PACIENT/MO_PR
МО прикрепления пациента, уже есть в табице cardz_clin, три последних цифры,
ставлю в пакет как f'250{d.mo_att}', если инокраевой, не заполняется (нет поля)
в XML.
xmlproc/utils
```py
_d["mo_pr"] = f'{REGION}{_d["mo_att"]}'
```
### ZL_LIST/ZAP/PACIENT/VZ
Вид занятости пациента, добавили поле VZ в таблицу cardz_clin, для
корректировки, если что, а вообще расчитывается автоматом по возрасту
- age < 25 - учащийся: 4
- age < 66 - раюотник: 1
- все остальные > 65 - пенсионер: 3
Если нужна корректировка то в приложении клиника делаем корректироваку.
xmlproc/utils

### ZL_LIST/ZAP/Z_SL/SL/PROFIL_M
Профиль Минздрава, поскольку все случаи профиль 76-рентгенология, ставлю
PROFIL_M=31-Радиология, радиотерапия (ХЗ?)
xmlstruct/hmxml
```py
'profil_m': 31,
```
### ZL_LIST/ZAP/Z_SL/SL/MOP
Где пациента приняли, тупо константа MOP=1, в поликлинике.
xmlstruct/hmxml
```py
'mop': 1,
```
### ZL_LIST/ZAP/Z_SL/SL/USL/LPU_1
ХЗ, тупо константа lpu_1='25202607000001001'
xmlstruct/hmxml
```py
'lpu_1': '25202607000001001',
```
